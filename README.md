# 机器学习新闻文本分类系统 python1028

## 项目概述

本项目是基于Python语言的机器学习新闻文本分类系统，采用Django框架进行后端开发，Vue.js和Node.js进行前端开发，实现了新闻文本的分类、管理以及用户登录和管理的功能。

## 技术栈

- **后端**
  - 开发语言：Python 3.7.7
  - 框架：Django
  - 数据库：MySQL 5.7+
  - 数据库工具：Navicat 11+

- **前端**
  - Node.js
  - Vue.js
  - HTML

## 功能模块

### 新闻分类模块

- 用户通过输入新闻标题和内容进行自动分类。
- 展示新闻分类的具体类别。
- 分类结果在文本框中显示。

### 新闻管理

- 管理已分类的新闻信息内容。
- 以列表形式展现新闻信息内容及分类结果。

### 用户管理

- 登录后实现系统功能使用。
- 允许用户修改个人密码和维护个人信息。

### 系统角色

- 普通用户：可进行新闻分类、查看新闻管理界面、管理个人信息。
- 管理员：除普通用户权限外，可管理用户信息，包括添加新用户和搜索用户。

## 运行环境

- 后端：Python 3.7.7、Django、MySQL 5.7+
- 前端：兼容现代浏览器的Node.js、Vue.js环境

## 部署步骤

1. 搭建Python开发环境，安装Python 3.7.7版本。
2. 安装Django框架。
3. 配置MySQL数据库。
4. 克隆前端代码，并在Node.js环境下编译。
5. 部署至服务器，确保端口和防火墙设置正确。
